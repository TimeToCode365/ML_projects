# Data Cleaning with Pandas

## Objective
Clean and preprocess a messy dataset to prepare it for analysis or modeling.

## Dataset
- Raw data: `customer_data.csv`
- Cleaned data: `Result_CleanedData.csv`

## Steps
1. Inspect data for missing values, duplicates, and inconsistencies
2. Handle missing values (imputation or removal)
3. Fix column formats (dates, numbers)
4. Remove duplicates
5. Save cleaned dataset

## Results
- Cleaned dataset ready for analysis or ML
- Observed patterns and issues summarized in notebook

## Skills Demonstrated
- Data exploration and cleaning using Pandas
- Handling missing values and duplicates
- Data preprocessing for ML
